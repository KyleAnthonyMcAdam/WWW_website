History of the internet

In 1962 the idea for a global computer network is released by MIT computer scientist J.C.R. Lickliter and in 1969 the first real wide-area computer network is created, linking Stanford and UCLA.

Fun Fact:

Another major inception in 1969 wass UNIX. UNIX is an operating system whose design formed the bases of Linux (one of the most popular operating systems used on modern web servers and web hosting services).

  • The beginning of the 1970s lead to small breakthroughs like the first email being sent in 1971, and also the birth of the eBook.
  • In 1973 the Transmission Control Protocol is created by Robert Kahn and Vinton Cerf. This technology allows multiple networks to link together without causing a cascade effect if one network collapses. Due to the increased popularity of email in 1975, the first email software is created. This software introducing the Reply and forward function of emails. With the growing popularity of the internet in 1977, the first PC modem is introduced into the commercial markets.

    • Did you know?

    The world’s first MMO (Massively multiplayer online) games emerged. Back then, they were called MUD’s (Multi-User Dungeon). These games were entirely text based fantasy games.

  • 3Com announces Ethernet products for both computer workstations and personal computers in 1981 and a year after that Telenet the first commercial network is created allowing for emails to be sent to multiple nations worldwide. Around the same time, the Domain name system is created with symbolics.com being the first registered domain.
  • 1990 marks the creation and implementation of URLs and HTML or hypertext markup language, this could create the first version of the world wide web.
  • 1991 brought some major innovations such as the first web page and in 1995 websites like amazon, yahoo, and eBay, as well as internet explores, are launched. During the same period, Java is created allowing for animations on websites.
  • To fight the growth of object-able material on the internet, the US Communication Decency Act is passed in congress in 1996.
  • In the 2000s google is dominating the search engine market but one of the biggest changes happened in 2009 when the US relaxed its control over ICANN the official naming body of the internet (domains)

    • Internet browsers

    A lot of new browsers were created and fought to be the dominant browser of choice and by the mid 1990’s the World Wide Web already had millions of active users. In 1995 The Software Tycoon Microsoft Corporation introduced Internet Explorer (IE) as an add-on with the new Windows 95 operating system. This made it the default browser on Windows operating systems and soon became the most popular Web browser in the world. Since then, a lot of Web browsers came and went such as Mozilla’s Firefox, Yahoo and in 2015 Microsoft replaced Internet Explorer with Edge. Surprisingly Google Chrome which was released in 2008 is still the most popular in 2021.
