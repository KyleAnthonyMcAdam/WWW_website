Client- and server-side engineering

Traditionally, a web developer would be responsible for building both the client-side and the server-side. This is called full-stack web development.

In recent years, however, this dynamic has changed. Though it certainly is still possible to become a full-stack web developer, the rise of new technologies for both front-end and back-end development, renders the task of becoming an expert in both areas virtually impossible. (Dhaduk, 2021)

As a result of this, large companies such as Google and Amazon typically hire employees to focus primarily on either client-side or server-side development. This ensures a higher quality product overall. (Dhaduk, 2021)

Technology used:

Client-side engineering typically makes use of:

  • HTML
  • CSS
  • JavaScript

As well as libraries such as:

  • JQuery
  • AngularJS
  • React Native
  • Bootstrap

The server-side is typically programed with:

  • PHP
  • Python
  • C#
  • JavaScript

And utilise data storage technologies such as:

  • MySQL
  • PostgreSLQ
  • MongoDB
  • Firebase
